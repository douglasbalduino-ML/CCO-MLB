Windows · Git & Bash
Configure Git e o terminal Bash
Cole o script abaixo no terminal do VSCode (Ctrl+`). Ele instala Git, gera sua chave SSH e configura o Git Bash como terminal padrão.
  • Abra o terminal do VSCode: Ctrl+`
  • Substitua SEU_NOME e seu.ldap e cole o script completo
  • Ao final, o terminal imprime sua chave pública SSH — copie ela
  • Acesse github.com → Settings → SSH Keys → New SSH Key e cole
  • Feche e reabra o VSCode — o terminal agora é Git Bash ✓
⚠ Após o winget install Git, feche e reabra o terminal antes de continuar — o PATH precisa ser recarregado.
📋 Erros comuns → Docs ↗
Script — Cole tudo de uma vez
# 1. Instalar Git
winget install --id Git.Git -e --source winget

# 2. Config — substitua os valores
git config --global user.name "SEU_NOME"
git config --global user.email "seu.ldap@mercadolivre.com"

# 3. Gerar chave SSH
ssh-keygen -t ed25519 -C "seu.ldap@mercadolivre.com" `
  -f "$env:USERPROFILE\.ssh\id_ed25519" -N ""

# 4. Ativar ssh-agent e adicionar chave
Get-Service ssh-agent | Set-Service -StartupType Automatic
Start-Service ssh-agent
ssh-add "$env:USERPROFILE\.ssh\id_ed25519"

# 5. Imprimir chave pública (copie abaixo)
Get-Content "$env:USERPROFILE\.ssh\id_ed25519.pub"

# 6. Configurar Git Bash como terminal do VSCode
$s = "$env:APPDATA\Code\User\settings.json"
$j = if (Test-Path $s) { Get-Content $s | ConvertFrom-Json } else { [PSCustomObject]@{} }
$j | Add-Member -Force NoteProperty "terminal.integrated.defaultProfile.windows" "Git Bash"
$j | ConvertTo-Json -Depth 10 | Set-Content $s
Write-Host "Git Bash configurado!" -ForegroundColor Green
🔑
Após o script: adicione a chave no GitHub
github.com → avatar → Settings → SSH and GPG keys → New SSH key → cole a chave impressa no terminal

Windows · Google
Configure o acesso Google
Para o Claude acessar Gmail, Google Sheets e BigQuery, autentique a CLI do Google direto pelo terminal do VSCode.
  • Abra o terminal: Ctrl+` (acento grave)
  • Se não tiver a CLI, rode no terminal: winget install --id Google.CloudSDK -e
  • Execute gcloud auth login — abre o browser, faça login com @mercadolibre.com
  • Execute gcloud auth application-default login — mesmos passos
  • Gmail, Sheets e BQ ficam acessíveis ao Claude ✓
ℹ O application-default login é necessário para que o Claude (e bibliotecas Python) acessem o BigQuery automaticamente.
PowerShell — Terminal VSCode (Ctrl+`)
PS> winget install --id Google.CloudSDK -e
Successfully installed Google Cloud SDK
PS> gcloud auth login
Opening browser for authorization...
You are now logged in as [seu@mercadolibre.com]
PS> gcloud auth application-default login
Credentials saved to AppData\gcloud\application_default_credentials.json
PS>

Tableau
Configure o token do Tableau
Para o Claude acessar dashboards e extracts do Tableau, gere um Personal Access Token (PAT) e envie para o Claude no chat.
  • Acesse us-east-1.online.tableau.com e faça login
  • Clique nas suas iniciais no canto superior direito
  • Selecione Configurações da Minha Conta
  • Localize o campo Tokens de Acesso Pessoal
  • Atribua um nome (ex: Claude) e clique em Gerar token
  • Copie o token e envie o nome + token no chat do Claude — ele faz a configuração
⚠ O token é exibido apenas uma vez. Copie antes de fechar a janela.

Configure o arquivo de contexto
Para que o Claude saiba quais tabelas usar, quais KPIs medir e como interpretar os dados do CCO FM, ele precisa de um arquivo CLAUDE.md.

  • O Claude lê esse arquivo automaticamente em toda nova sessão
  • Use Ctrl+L para abrir o chat e começar a conversar
